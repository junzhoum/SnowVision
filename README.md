# SnowVision
A project of exploring culture heritage objects with computer vision approches

### Dependency:
- caffe==1.0.0
- pcl==1.8.0
- opencv==3.4.1
- skimage==0.10.1

### How to use:
1. make (If everything goes well, it will output the library "sherd2depth.so")
2. python match.py (Results are saved in ./data)
